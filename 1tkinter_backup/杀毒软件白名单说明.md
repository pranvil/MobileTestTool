## 🛡️ 常见杀毒软件设置方法：

### Windows Defender
1. 打开"Windows安全中心"
2. 点击"病毒和威胁防护"
3. 点击"病毒和威胁防护设置"
4. 点击"添加或删除排除项"
5. 添加上述文件夹路径

### 360安全卫士
1. 打开360安全卫士
2. 点击"木马防火墙"
3. 点击"信任与阻止"
4. 点击"信任区"
5. 添加上述文件夹路径

### 腾讯电脑管家
1. 打开腾讯电脑管家
2. 点击"病毒查杀"
3. 点击"信任区"
4. 添加上述文件夹路径

### 火绒安全
1. 打开火绒安全
2. 点击"防护中心"
3. 点击"信任区"
4. 添加上述文件夹路径
